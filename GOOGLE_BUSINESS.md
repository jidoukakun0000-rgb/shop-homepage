# Googleビジネスプロフィール対応メモ（陽気軒）

> 広告運営にあたり、Googleマップ／Googleビジネスプロフィールの設定を進めるためのメモ。
> 関連: GitHub issue #6

## 最終更新

2026-06-03

## 現状の問題

1. **ホームページがGoogleマップに表示されない**
   - Googleマップ上に店舗は出ているが、ホームページURLが連携・表示されていない
2. **営業時間を設定できない**
   - Googleマップの店舗情報に営業時間を設定しようとしても設定できない

### 原因

現状は「誰も管理していない／不明」な状態で、**Googleビジネスプロフィールの「オーナー確認（認証）」が済んでいない**ことが根本にある。

> ⚠️ **重要な注意（公式仕様で確認済み）**
> 「未認証だと一切編集できない」わけではない。Google公式ヘルプ上、**未認証でも営業時間・ウェブサイトURL・名前・住所・電話番号などの基本情報は編集できる**。
> ただし、(1) すべての項目の編集、(2) 編集内容が検索・マップへ確実に公開反映されること、(3) クチコミ返信・投稿・インサイト等の全機能の利用 ——— これらには**オーナー確認が必要**。
>
> したがって「営業時間を設定できない」原因は、未認証そのものとは限らず、以下も疑うべき：
> - 既に**別の誰かがオーナー登録**していて編集が上書きできない
> - 編集はできているが**公開反映待ち／審査中**
> - 入力経路を間違えている（マップアプリではなくビジネスプロフィール側で編集する必要）
>
> いずれにせよ、確実に運用するにはオーナー確認の取得が前提になる。

## 対応方針：オーナー確認を申請する

運営担当（こちら）が主導してオーナー確認を申請する。最終的なオーナーは店主に残す（下記アカウント構成を参照）。

### ① 準備するもの
- [ ] 運営用のGoogleアカウント（お店専用が望ましい）
- [ ] お店の情報（正式名称 / 住所 / 電話番号 / 業種カテゴリ）
- [ ] サイトURL: https://jidoukakun0000-rgb.github.io/shop-homepage/youkiken/

### ② 店舗を検索して申請
1. **https://business.google.com/add** にアクセス（または Googleマップ／Google検索でお店を検索）
2. お店の名前・住所で検索 → リスト/マップに出ている店舗を選ぶ
3. 「**このビジネスを管理（Claim this business）**」→「**今すぐ管理（Manage now）**」をクリック
   - 既に他人が登録済みの場合は「**他のユーザーが管理している可能性があります**」と表示され、「**アクセスをリクエスト（Request access）**」になる（→ 後述の注意点）
4. オーナー確認（認証）の手続きに進む

### ③ オーナー確認の方法（★Googleが自動で決める。ユーザーは選べない）

> ⚠️ 確認方法は**業種・地域・営業時間・公開情報をもとにGoogleが自動的に決定**し、**原則ユーザーは変更・選択できない**。複数方式の併用を求められることもある。

| 方法 | 内容 | 所要時間 |
|---|---|---|
| 動画録画 | 店の看板→店内→レジ/厨房など実在を示す動画を撮影・提出 | 数日で審査 |
| ライブ動画通話 | Google担当者と通話しながら店内を見せる（営業時間内・一部業種のみ） | 当日〜 |
| 電話/SMS | 登録電話番号にコード送信（IVR・自動応答は不可） | 即時 |
| メール | プロフィールに表示されたアドレスにコード（独自ドメイン等） | 即時 |
| ハガキ | 店舗住所に確認コードが郵送（約14日、コード有効30日） | 数日〜2週間 |
| Search Console連携 | 同一ドメインを認証済みなら即時確認できる場合あり | 即時 |

> 📌 **2025〜2026年のトレンド**：偽リスティング対策で**動画確認が主流化・標準化**しており、特に新規店舗では動画が優先提示され、**ハガキは縮小傾向**。ただし全業種一律の必須化ではなく、店舗により電話/SMS/メールが提示される。**Googleが提示した方式に従うのが最短**。

### ④ 確認が完了すると「確実に公開・全機能利用」できること
- [ ] 営業時間の設定が**確実に反映**される（未認証でも編集自体は可能だが反映が不安定）
- [ ] ホームページURLの登録が反映される
- [ ] 写真・説明・メニュー等のすべての編集
- [ ] クチコミ返信・投稿・インサイト（パフォーマンス分析）などの全機能

## SMS/電話認証はどこに届くか

- SMS/電話認証のコードは、Googleビジネスプロフィールに**登録した電話番号**宛てに、自動音声通話またはSMSで届く。
  - 登録番号が**店の固定電話**なら → 店に届く（店主が受け取る）
  - 登録番号に**運営者の携帯**を設定していれば → その携帯に届く
- つまり「店舗の番号か個人携帯か」は**登録内容次第**。「個人携帯には絶対届かない」わけではない。
- ⚠️ **IVR（自動音声応答）や自動応答ではコードを取得できない**。人が応答できる番号にすること。
- 運用上は、店の固定電話を登録番号にしておき、**店主が受け取った数字を運営に伝える**のがシンプル（店主は数字を読み上げるだけ）。

## 【重要】アカウント構成

Googleの役割は**3種類**ある：**メインのオーナー（primary owner）／オーナー（owner）／管理者（manager）**。
最終的な支配権（権限の譲渡・プロフィール削除・他ユーザーの追加削除）は**「メインのオーナー」だけ**が持つ（1プロフィールに1人のみ）。管理者はほぼ同等の編集ができるが、ユーザー追加削除・プロフィール削除はできない。

| 役割 | 誰のアカウント | 権限 |
|---|---|---|
| **メインのオーナー** | **店主**のGoogleアカウント | 最終支配権。権限譲渡・削除・ユーザー管理が可能 |
| オーナー | （必要なら）信頼できる関係者 | ほぼ全権だがメインオーナーの座は持たない |
| 管理者（実作業） | **運営**のアカウント | 営業時間・URL・写真などの編集は可能。ユーザー追加削除・削除は不可 |

- 店主は単なる「オーナー」ではなく必ず**「メインのオーナー」**にする。運営は**「管理者」**で十分に実作業ができる。
- **運営をメインオーナーにしない**こと。メインオーナー権限の譲渡は**本人しか実行できない**ため、契約終了時に運営が協力しないと店主へ権限を戻せず揉める（最終的にGoogleへの異議申し立てで回収しうるが手間と時間がかかる）。
- 所有権を移譲する場合、クチコミ等のデータは維持されるが**7日間の保留期間**がある。
- 店主が本当に何もできない場合は、運営アカウントで一旦すべてやり**後で店主のアカウントをメインオーナーに移譲する**運用も可。最終的に店主へ移すことを前提にする。

## 店主への説明文（そのまま渡せる）

> **【お店のGoogle対応について】**
>
> お店をGoogleマップで調べると、今は「名前と場所」だけが出ている状態です。これは"仮の登録"のようなもので、誰も正式な持ち主として登録していません。
>
> このままだと、
> - **営業時間が表示されない**（お客様が「やってるの？」と分からない）
> - **ホームページが出てこない**（メニューや雰囲気が伝わらない）
> - **写真やお店の説明を載せられない**
>
> そこで、**「このお店の正式な持ち主は私たちです」とGoogleに登録（オーナー確認）** します。これをやると上のことが全部できるようになり、検索で見つけてもらいやすくなって**集客につながります**。
>
> 登録の途中で、Googleから**お店の電話番号あてに、ショートメール（SMS）か自動音声で数字のコードが届く**ことがあります。届いたら、その**数字を私（運営担当）に教えてください**。それだけで大丈夫です。
> （※お店の固定電話を登録するので、コードはお店に届きます。動画での確認をお願いされる場合もあります）

## 注意点

- 申請前に、**既に他の誰か（前の運営者・元オーナー等）がオーナー登録していないか**を確認する。
- 既存オーナーがいる場合、「**他のユーザーが管理している可能性があります（Someone else manages this business on Google）**」と表示され、**そのオーナーへのアクセス権リクエスト（Request access）**になる。
  - 現オーナーには通知メールが届き、**3日以内**に承認/拒否する必要がある。
  - 3日経っても応答がない場合、自動で権限が移るわけではなく、確認メールの「リクエストを表示」→「確認」から**自分でオーナー確認手続きに進める場合がある**（条件付き）。拒否・無応答時は**再審査請求（appeal）**も可能。非店舗型などはGoogleへの直接問い合わせが必要なこともある。
- まずは心当たりの関係者（前の制作者・元オーナー等）に許可してもらうのが最短。
