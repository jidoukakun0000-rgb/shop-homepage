## 明日（2026-06-05）の作業：メニュー写真の撮り直し＆照合

別セッションで実施予定。実物メニューを撮り直し、`youkiken/menu.json` と突き合わせて確定する。

### 1. 撮影（ラミネート対策：真上から水平・反射が価格欄に乗らない角度で）
- 平日・土日それぞれ **表・裏の計4枚**（全体）＋ 下記⚠️箇所の**寄り**も撮る
- 撮ったら `youkiken/menu-src/` 等に保存し、ファイルパスをセッションに渡す（貼付画像はサブエージェントが読めないため**ファイル必須**）

### 2. 要確認（⚠️＝今の menu.json で未確定 or 推測の箇所）
- [ ] **塩ラーメン（しお）**の品目・並び・価格（「玉子あんかけ¥550」が不自然で現状は省略中）
- [ ] **定食末尾の赤字**（半ねぎラーメン定食／半ギョーハン／半チャーハン／半ギョーザ2個）の価格 ＝ 判読不能で現状未掲載
- [ ] **単品レバニラ炒め**の価格（¥700 か ¥750）
- [ ] **ネギ玉炒め**¥750 / **ジャンボギョーザ定食**¥950（手書き訂正）が正しいか
- [ ] **大盛りルール**：ラーメン大盛り＝平日無料／土日祝＋¥200、ご飯類大盛り＝両日¥200 で正しいか
- [ ] **税込/税別**の表記が実際にあるか（現状「価格はすべて税込」と注記）

### 3. 照合〜反映の手順
1. 4枚を**複数エージェント＋自分の目視**で転記・差分照合（※画像のOCRはハルシネーションが出やすいので主は人間の精読）
2. 確定値で `youkiken/menu.json` を編集（構造：`days.weekday|weekend.categories[].items[]{name,price,tags}` ＋ `setInfo` ＋ `globalNotes`）
3. `python3 -m http.server` でローカル確認 → `git commit && git push origin main`（Pages自動反映）
- 平日＝フルメニュー / 土日＝簡略版（定食→単品、品数減）。共通品の価格は同一。
- 人気タグ＝陽気ラーメン、名物タグ＝ジャンボ餃子。
